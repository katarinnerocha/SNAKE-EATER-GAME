JOGO DA COBRA

Inpirado nos jogos classicos da Cobra em Arcade.

COMANDOS:
- Para movimentar a COBRE use as setas: CIMA, BAIXO, ESQUERDA e DIREITA.

REGRAS:
- Colete os QUADRADOS VERMELHOS que aparencem de forma aleatoria no cenário para ganhar PONTOS (SCORE);
- A cada coleta de QUADRADOS VERMELHOS é adicionado 1 quadrado na COBRA e 1 ponto no SCORE; 
- Os PONTOS (SCORE) estão no topo da tela;
- Se a COBRA chocar nas paredes, é GAME OVER;
- Se a COBRA chocar em si, é GAME OVER;


SNAKE GAME

Inspired by classic Snake arcade games.

CONTROLS:
- To move the SNAKE, use the arrow keys: UP, DOWN, LEFT, and RIGHT.

RULES:
- Collect the RED SQUARES that appear randomly on the screen to earn POINTS (SCORE);
- Each collection of RED SQUARES adds 1 square to the SNAKE and 1 point to the SCORE;
- The POINTS (SCORE) are at the top of the screen;
- If the SNAKE collides with the walls, it's GAME OVER;
- If the SNAKE collides with itself, it's GAME OVER;
